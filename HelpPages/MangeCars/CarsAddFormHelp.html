مساعدة شاشة إضافة سيارة

هذه الشاشة مخصصة لإضافة سيارة جديدة إلى النظام، حيث يمكن إدخال بيانات السيارة الأساسية مثل المعرف، البراند، الموديل، الفئة، نوع الجير، نوع الوقود، سعر الإيجار اليومي، سنة الصنع، سعر السيارة، واللون.

أولاً: الحقول الأساسية

  • المعرف: للقراءة فقط، يتم توليده تلقائياً من قاعدة البيانات.
  • البراند: اسم الشركة المصنعة للسيارة.
  • الموديل: موديل السيارة.
  • الفئة: فئة السيارة (مثل SUV، سيدان، إلخ).
  • نوع الجير: أوتوماتيك أو يدوي.
  • نوع الوقود: بنزين، ديزل، كهرباء، إلخ.
  • سعر الإيجار اليومي: السعر الذي يؤجر به اليوم الواحد.
  • سنة الصنع: سنة تصنيع السيارة.
  • سعر السيارة: السعر الكامل للسيارة.
  • اللون: لون السيارة.

ثانياً: الأزرار

  • زر حفظ: لحفظ بيانات السيارة المدخلة.
  • زر المساعدة: لعرض هذه الصفحة التعليمية.
  • زر الرجوع: للعودة إلى شاشة إدارة السيارات.

ملاحظات عامة

  • تأكد من إدخال جميع البيانات المطلوبة قبل الحفظ.
  • المعرف يتم توليده تلقائياً ولا يمكن تغييره.

Add Car Screen Help

This screen is used to add a new car to the system, where you can enter basic car information such as ID, brand, model, category, gear type, fuel type, daily rental price, manufacturing year, car price, and color.

First: Basic Fields

  • ID: Read-only, auto-generated from the database.
  • Brand: Car manufacturer.
  • Model: Car model.
  • Category: Car category (e.g., SUV, Sedan, etc.).
  • Gear Type: Automatic or Manual.
  • Fuel Type: Petrol, Diesel, Electric, etc.
  • Daily Rental Price: Rental price per day.
  • Manufacturing Year: Car manufacturing year.
  • Car Price: Full price of the car.
  • Color: Car color.

Second: Buttons

  • Save Button: Saves the entered car data.
  • Help Button: Displays this help page.
  • Back Button: Returns to the car management screen.

General Notes

  • Ensure all required data is entered before saving.
  • ID is auto-generated and cannot be changed.
