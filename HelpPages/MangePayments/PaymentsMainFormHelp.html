مساعدة شاشة إدارة المدفوعات

هذه الشاشة مخصصة لإدارة جميع عمليات الدفع داخل النظام، حيث يمكن عرض جميع المدفوعات، البحث عن عملية دفع محددة، إضافة مدفوعات جديدة، تعديل المدفوعات، وطباعة التقارير.

أولاً: جدول عرض المدفوعات

يعرض هذا الجدول جميع المدفوعات المسجلة في النظام، ويمكن استخدامه أيضاً لعرض نتائج البحث.

عملية البحث

  • يمكن البحث عن عملية دفع باستخدام المعرف.
  • اضغط زر بحث لعرض النتائج في الجدول.
  • زر إعادة التعيين يعيد الجدول إلى عرض جميع المدفوعات بعد البحث.

ثانياً: الأزرار الأساسية

  • زر المساعدة: لعرض هذه الصفحة التعليمية.
  • زر إضافة عملية دفع: لفتح شاشة إدخال عملية دفع جديدة.
  • زر إدارة عملية دفع: لتعديل عملية الدفع المحددة في الجدول.
  • زر الطباعة: لطباعة قائمة بجميع المدفوعات.
  • زر الخروج: لإغلاق البرنامج والخروج.

ملاحظات عامة

  • تأكد من تحديد عملية الدفع قبل استخدام زر إدارة عملية دفع.
  • زر البحث يسهل العثور على عملية دفع محددة بسرعة.
  • يجب حفظ أي تعديلات قبل الخروج من البرنامج.

Payments Management Screen Help

This screen is used to manage all payments in the system, where you can view all payments, search for a specific payment, add new payments, edit payments, and print reports.

First: Payments Table

This table displays all registered payments in the system, and can also be used to show search results.

Search Function

  • You can search for a payment using ID.
  • Click the Search button to display results in the table.
  • The Reset button restores the table to show all payments after a search.

Second: Main Buttons

  • Help Button: Displays this help page.
  • Add Payment Button: Opens the Add Payment screen.
  • Edit Payment Button: Edit the selected payment in the table.
  • Print Button: Prints a list of all payments.
  • Exit Button: Closes the application.

General Notes

  • Ensure a payment is selected before using the Edit Payment button.
  • The search button helps quickly locate a specific payment.
  • Any changes should be saved before exiting the application.
