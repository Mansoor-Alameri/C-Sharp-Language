مساعدة شاشة إضافة عميل

هذه الشاشة مخصصة لإضافة عميل جديد للنظام، حيث يمكن إدخال البيانات الأساسية للعميل مثل الاسم، العنوان، رقم الهاتف، نوع العميل، البطاقة التعريفية، ورقم رخصة القيادة، وغيرها.

أولاً: الحقول الأساسية

  • ID العميل: للقراءة فقط، يتم توليده تلقائياً من النظام.
  • الاسم: الاسم الأول و الأخير للعميل.
  • رخصة القيادة: رقم رخصة قيادة العميل.
  • العنوان: عنوان السكن الحالي للعميل.
  • نوع الضمانة: نقدي، عقاري، أو أخرى.
  • نوع العميل: فرد أو شركة.
  • نوع البطاقة التعريفية: جواز سفر أو بطاقة قومية.
  • قيمة البطاقة التعريفية: رقم أو قيمة البطاقة.
  • الجنس: ذكر أو أنثى.
  • رقم الهاتف الأول: إجباري.
  • رقم الهاتف الثاني: اختياري.
  • تاريخ الميلاد: السنة فقط.

ثانياً: الأزرار

  • زر الرجوع: للعودة إلى الشاشة الرئيسية لإدارة العملاء.
  • زر الحفظ: لحفظ بيانات العميل المدخلة.
  • زر المساعدة: لعرض هذه الصفحة التعليمية.

ملاحظات عامة

  • تأكد من إدخال جميع البيانات المطلوبة قبل الحفظ.
  • يجب ملء رقم الهاتف الأول obligatorily.
  • يمكن ترك رقم الهاتف الثاني فارغاً إذا لم يكن موجوداً.




Add Client Screen Help

This screen is designed to add a new client to the system, where you can enter the client's basic information such as name, address, phone number, client type, ID type, driving license, and more.

First: Basic Fields

  • ID: Read-only, auto-generated by the system.
  • Name: First and last name of the client.
  • Driving License: Client's driving license number.
  • Address: Client's current address.
  • Guarantee Type: Cash, Property, or Other.
  • Customer Type: Individual or Company.
  • Identifiy Type: Passport or National ID.
  • Identifiy: ID number or value.
  • Gender: Male or Female.
  • Phone 1: Required.
  • Phone 2: Optional.
  • Date Of Birth: Year only.

Second: Buttons

  • Back Button: Returns to the main customer management screen.
  • Save Button: Saves the entered client data.
  • Help Button: Displays this help page.

General Notes

  • Ensure all required fields are filled before saving.
  • Phone 1 must be filled obligatorily.
  • Phone 2 can be left empty if not available.
