مساعدة شاشة إدارة العملاء

هذه الشاشة مخصصة لإدارة بيانات العملاء داخل النظام، وتُستخدم لعرض جميع العملاء، البحث عن عميل معيّن، وإجراء العمليات الأساسية مثل الإضافة، التعديل، والطباعة.

أولاً: جدول عرض العملاء

يعرض هذا الجدول جميع العملاء المسجلين في النظام. كما يتم استخدامه لعرض نتائج البحث عند تنفيذ عملية البحث.

  • يتم تحديد العميل بالنقر على الصف الخاص به.
  • يجب تحديد عميل قبل استخدام زر "إدارة عميل".

ثانياً: البحث عن عميل

تتيح لك منطقة البحث البحث عن عميل باستخدام البيانات المتاحة.

  • أدخل بيانات البحث المطلوبة.
  • اضغط زر بحث لعرض النتائج في الجدول.
  • يتم تحديث الجدول تلقائياً بنتائج البحث.

زر إعادة التعيين

زر إعادة التعيين يستخدم لإلغاء عملية البحث وإعادة عرض جميع العملاء في الجدول.

ثالثاً: القائمة الجانبية

زر إضافة عميل

يفتح هذا الزر شاشة إضافة عميل جديد، ويُستخدم لإدخال بيانات عميل لأول مرة.

زر إدارة عميل

يفتح هذا الزر شاشة تعديل بيانات العميل المحدد في الجدول.

ملاحظة: يجب تحديد عميل من الجدول قبل استخدام هذا الزر.

زر طباعة

يُستخدم هذا الزر لطباعة تقرير يحتوي على جميع العملاء وبياناتهم الحالية المعروضة في الجدول.

زر المساعدة

يعرض هذه الصفحة التي تشرح كيفية استخدام شاشة إدارة العملاء.

زر الخروج

يُستخدم لإغلاق البرنامج والخروج منه بشكل كامل.

ملاحظات عامة

  • تأكد من إدخال بيانات صحيحة عند البحث.
  • في حال عدم ظهور نتائج، قد لا يكون العميل موجوداً.
  • تأكد من حفظ أي تعديلات قبل الخروج من البرنامج.



Customer Management Screen Help

This screen is designed to manage customer data within the system. It is used to display all customers, search for a specific customer, and perform basic operations such as adding, editing, and printing.

First: Customer List Table

This table displays all customers registered in the system. It is also used to show search results when a search operation is performed.

  • A customer is selected by clicking on the corresponding row.
  • A customer must be selected before using the "Manage Customer" button.

Second: Customer Search

The search section allows you to find a customer using the available search criteria.

  • Enter the required search information.
  • Click the Search button to display the results in the table.
  • The table is automatically updated with the search results.

Reset Button

The Reset button is used to cancel the search operation and display all customers again in the table.

Third: Side Menu

Add Customer Button

This button opens the Add Customer screen, which is used to enter a new customer's data for the first time.

Manage Customer Button

This button opens the Edit Customer screen for the customer currently selected in the table.

Note: A customer must be selected from the table before using this button.

Print Button

This button is used to print a report containing all customers and their currently displayed information.

Help Button

This button displays this help page, which explains how to use the Customer Management screen.

Exit Button

This button is used to close the application and exit the system completely.

General Notes

  • Make sure to enter correct information when searching.
  • If no results appear, the customer may not exist.
  • Ensure that any changes are saved before exiting the application.
