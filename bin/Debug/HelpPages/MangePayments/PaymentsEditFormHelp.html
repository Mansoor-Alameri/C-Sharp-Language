مساعدة شاشة تعديل عملية دفع

هذه الشاشة مخصصة لتعديل عملية دفع موجودة داخل النظام، حيث يمكن تعديل جميع البيانات الأساسية للعملية المدخلة مسبقاً.

أولاً: الحقول الأساسية

  • المعرف: للقراءة فقط، يتم توليده تلقائياً ولا يمكن تغييره.
  • معرف عملية الإيجار: الرقم المرجعي لعملية الإيجار المرتبطة بالدفع.
  • تاريخ الدفع: اليوم الذي تمت فيه عملية الدفع.
  • نوع الدفعة: يمكن أن تكون للإيجار أو لمخالفة.
  • طريقة الدفع: مثل نقدي، بطاقة، أو تحويل.
  • الخصم: للقراءة فقط، يتم حسابه تلقائياً.
  • السعر الكلي: للقراءة فقط، يتم حسابه تلقائياً.

ثانياً: الأزرار

  • زر حفظ: لحفظ التعديلات على عملية الدفع.
  • زر المساعدة: لعرض هذه الصفحة التعليمية.
  • زر الرجوع: للعودة إلى شاشة المدفوعات.
  • زر الحذف: لحذف عملية الدفع المحددة.

ملاحظات عامة

  • تأكد من تحديد عملية الدفع قبل استخدام زر تعديل أو حذف.
  • الخصم والسعر الكلي يتم حسابهما تلقائياً ولا يمكن تعديلهما يدوياً.
  • تأكد من إدخال تاريخ الدفع بشكل صحيح.

Edit Payment Screen Help

This screen is used to edit an existing payment in the system, where all essential fields of the previously entered payment can be modified.

First: Basic Fields

  • ID: Read-only, auto-generated and cannot be changed.
  • Rental ID: Reference number of the related rental.
  • Payment Date: The date the payment was made.
  • Payment Type: Can be for rental or fine.
  • Payment Method: e.g., Cash, Card, or Transfer.
  • Discount: Read-only, calculated automatically.
  • Total Amount: Read-only, calculated automatically.

Second: Buttons

  • Save Button: Saves the changes to the payment.
  • Help Button: Displays this help page.
  • Back Button: Returns to the Payments screen.
  • Delete Button: Deletes the selected payment.

General Notes

  • Ensure a payment is selected before using Edit or Delete buttons.
  • Discount and total amount are calculated automatically and cannot be edited manually.
  • Ensure the payment date is entered correctly.
