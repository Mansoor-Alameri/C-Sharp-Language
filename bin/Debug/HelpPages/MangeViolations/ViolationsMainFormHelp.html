مساعدة شاشة إدارة المخالفات

هذه الشاشة مخصصة لإدارة المخالفات المرتبطة بإيجار محدد، حيث يمكن عرض المخالفات، إضافتها، دفعها، وطباعة تقارير عنها.

أولاً: جدول عرض المخالفات

يعرض هذا الجدول جميع المخالفات المرتبطة بالإيجار المختار.

حقول البحث والتحكم في الجدول

  • حقل معرف الإيجار لإدخال رقم الإيجار المطلوب عرض مخالفاته.
  • زر عرض لعرض المخالفات المرتبطة بالإيجار.
  • زر إعادة التعيين لإفراغ الجدول وإعادة الحقل إلى الوضع الفارغ.

الحقول أسفل الجدول

  • المجموع: يظهر المبلغ الإجمالي لجميع المخالفات.
  • المبلغ المدفوع: يظهر المبلغ المدفوع من المخالفات.
  • المبلغ المتبقي: يظهر المبلغ المتبقي من المخالفات غير المدفوعة.

ثانياً: الأزرار

  • زر المساعدة: لعرض هذه الصفحة التعليمية.
  • زر إضافة مخالفة: لإدخال مخالفة جديدة.
  • زر الدفع: يتم تفعيله فقط إذا كانت هناك مخالفات غير مدفوعة.
  • زر الطباعة: لطباعة المخالفات الخاصة بالإيجار المحدد.
  • زر الخروج: لإغلاق البرنامج والخروج.

ملاحظات عامة

  • تأكد من إدخال معرف إيجار صحيح قبل عرض المخالفات.
  • زر الدفع يكون مفعلاً فقط عند وجود مخالفات غير مدفوعة.
  • تأكد من مراجعة الحقول أسفل الجدول لمعرفة المبالغ المدفوعة والمتبقية.

Fines Management Screen Help

This screen is used to manage fines associated with a specific rental, where you can view fines, add new fines, make payments, and print reports.

First: Fines Table

This table displays all fines associated with the selected rental.

Search and Table Controls

  • Rental ID: Enter the rental number to display its fines.
  • Show Button: Displays fines linked to the entered rental.
  • Reset Button: Clears the table and resets the input field.

Fields Below the Table

  • Total: Displays the total amount of all fines.
  • Paid Amount: Shows the amount already paid.
  • Remaining Amount: Shows the remaining unpaid fines.

Second: Buttons

  • Help Button: Displays this help page.
  • Add Fine Button: Adds a new fine.
  • Pay Button: Enabled only if there are unpaid fines.
  • Print Button: Prints fines of the selected rental.
  • Exit Button: Closes the application.

General Notes

  • Ensure a valid Rental ID is entered before displaying fines.
  • The Pay button is enabled only if there are unpaid fines.
  • Check the fields below the table to see paid and remaining amounts.
