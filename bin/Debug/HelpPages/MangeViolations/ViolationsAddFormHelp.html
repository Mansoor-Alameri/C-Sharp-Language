مساعدة شاشة إضافة مخالفة

هذه الشاشة مخصصة لإدخال مخالفة جديدة مرتبطة بإيجار محدد.

أولاً: الحقول الأساسية

  • المعرف: للقراءة فقط، يتم توليده تلقائياً.
  • معرف الإيجار: الرقم المرجعي لعملية الإيجار المرتبطة بالمخالفة.
  • المخالفة: وصف المخالفة.
  • سعر المخالفة: قيمة المخالفة المالية.

ثانياً: الأزرار

  • زر حفظ: لحفظ المخالفة الجديدة.
  • زر المساعدة: لعرض هذه الصفحة التعليمية.
  • زر الرجوع: للعودة إلى شاشة إدارة المخالفات.

ملاحظات عامة

  • تأكد من إدخال معرف إيجار صحيح قبل حفظ المخالفة.
  • يتم توليد المعرف تلقائياً ولا يمكن تغييره يدوياً.

Add Fine Screen Help

This screen is used to enter a new fine associated with a specific rental.

First: Basic Fields

  • ID: Read-only, auto-generated.
  • Rental ID: Reference number of the rental linked to this fine.
  • Fine Description: Description of the fine.
  • Fine Amount: Monetary value of the fine.

Second: Buttons

  • Save Button: Saves the new fine.
  • Help Button: Displays this help page.
  • Back Button: Returns to the Fines Management screen.

General Notes

  • Ensure a valid Rental ID is entered before saving the fine.
  • The ID is auto-generated and cannot be edited manually.
