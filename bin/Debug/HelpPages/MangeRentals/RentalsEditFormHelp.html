مساعدة شاشة تعديل إيجار

هذه الشاشة مخصصة لتعديل بيانات إيجار موجود، ويمكن تعديل جميع الحقول فقط إذا كان الإيجار لم يتم دفع قيمته بعد. الإيجارات المدفوعة لا يمكن تعديلها.

أولاً: الحقول الأساسية

  • المعرف: للقراءة فقط، يتم جلبه من قاعدة البيانات.
  • العميل: يمكن تعديله.
  • السيارة: يمكن تعديلها إذا كان الإيجار غير مدفوع.
  • رقم العقد الموقع من العميل: يمكن تعديله.
  • الخصم: يمكن تعديله.
  • سعر الإيجار اليومي: يمكن تعديله.
  • السعر الكلي: يتم حسابه تلقائياً بعد أي تعديل.
  • تاريخ بداية الإيجار: يمكن تعديله.
  • تاريخ نهاية الإيجار: يمكن تعديله.
  • عدد الأيام: يتم تحديثه تلقائياً بعد تعديل التواريخ.
  • الملاحظات: يمكن إضافة أي ملاحظات تتعلق بالإيجار.
ملاحظة: لا يمكن تعديل الإيجار إذا كان قد تم دفع قيمته بالفعل. زر الحذف متاح فقط للإيجارات غير المدفوعة.

ثانياً: الأزرار

  • زر حفظ: لحفظ التعديلات على الإيجار.
  • زر المساعدة: لعرض هذه الصفحة التعليمية.
  • زر الرجوع: للعودة إلى شاشة إدارة الإيجارات.
  • زر الحذف: لحذف الإيجار إذا لم يتم دفع قيمته.

ملاحظات عامة

  • تأكد من أن الإيجار غير مدفوع قبل تعديل أي بيانات أو الحذف.
  • المعرف يتم جلبه من قاعدة البيانات ولا يمكن تغييره.
  • السعر الكلي وعدد الأيام يتم حسابهما تلقائياً بعد تعديل التواريخ أو السعر اليومي.

Edit Rental Screen Help

This screen is used to edit an existing rental. All fields can be modified only if the rental has not been paid yet. Paid rentals cannot be edited.

First: Basic Fields

  • Rental ID: Read-only, fetched from the database.
  • Client: Can be edited.
  • Car: Can be changed if the rental is unpaid.
  • Contract Number: Can be modified.
  • Discount: Can be edited.
  • Daily Rental Price: Can be modified.
  • Total Price: Automatically calculated after any changes.
  • Start Date: Editable.
  • End Date: Editable.
  • Number of Days: Automatically updated after modifying dates.
  • Notes: Any notes regarding the rental can be added.
Note: Rentals that have already been paid cannot be edited. The delete button is available only for unpaid rentals.

Second: Buttons

  • Save Button: Saves the changes to the rental.
  • Help Button: Displays this help page.
  • Back Button: Returns to the rental management screen.
  • Delete Button: Deletes the rental if it has not been paid.

General Notes

  • Ensure the rental is unpaid before editing or deleting.
  • Rental ID is fetched from the database and cannot be changed.
  • Total price and number of days are automatically calculated after modifying dates or daily price.
