مساعدة شاشة إدارة الإيجارات

هذه الشاشة مخصصة لإدارة جميع الإيجارات داخل النظام، حيث يمكن عرض جميع الإيجارات، البحث عن إيجار محدد، إضافة إيجار جديد، تعديل بيانات الإيجار، وطباعة تقارير الإيجارات.

أولاً: جدول عرض الإيجارات

يعرض هذا الجدول جميع الإيجارات في النظام. يمكن استخدامه أيضاً لعرض نتائج البحث أو الفلترة.

فلتر الإيجارات

يوجد كومبو بوكس باسم فلتر يسمح بتصفية الجدول حسب:

  • كل الإيجارات
  • الإيجارات المنتهية
  • الإيجارات الحالية

أزرار البحث وإعادة التعيين

  • زر البحث: للبحث عن الإيجارات باستخدام البيانات المدخلة.
  • زر إعادة التعيين: يعيد الجدول إلى عرض كل الإيجارات بعد البحث أو الفلترة.

ثانياً: الأزرار الأساسية للشاشة

  • زر المساعدة: لعرض هذه الصفحة التعليمية.
  • زر إضافة إيجار: لفتح شاشة إدخال إيجار جديد.
  • زر إدارة إيجار: لتعديل بيانات الإيجار المحدد، متاح فقط للإيجارات غير المدفوعة ويجب أن يكون هناك اختيار على الصف.
  • زر الدفع: يكون متاحاً فقط إذا كان الإيجار يحتوي على مخالفات غير مدفوعة أو الإيجار نفسه غير مدفوع.
  • زر إطلاق/تحرير السيارة: لتحرير السيارة المرتبطة بالإيجار المنتهي وجعلها متاحة للإيجار.
  • زر المخالفات: يفتح فورم عرض المخالفات الخاصة بهذا الإيجار.
  • زر الطباعة: لطباعة كل الإيجارات.
  • زر الخروج: لإغلاق البرنامج والخروج.

ملاحظات عامة

  • تأكد من تحديد صف الإيجار قبل استخدام أي زر يتطلب اختيار.
  • زر الدفع يكون متاح فقط في الحالات المحددة.
  • يمكن استخدام الفلتر لتسهيل البحث عن الإيجارات المنتهية أو الحالية.

Rental Management Screen Help

This screen is designed to manage all rentals in the system, where you can view all rentals, search for a specific rental, add a new rental, edit rental data, and print rental reports.

First: Rentals Table

This table displays all rentals in the system. It is also used to show search results or filtered rentals.

Rental Filter

There is a combo box named Filter that allows filtering the table by:

  • All rentals
  • Expired rentals
  • Current rentals

Search and Reset Buttons

  • Search Button: To search rentals using the entered data.
  • Reset Button: Resets the table to show all rentals after search or filtering.

Second: Main Screen Buttons

  • Help Button: Displays this help page.
  • Add Rental Button: Opens the Add Rental screen.
  • Manage Rental Button: Edit the selected rental; only enabled for unpaid rentals and requires a selection.
  • Payment Button: Enabled only if the rental has unpaid fines or is unpaid.
  • Release/Free Car Button: Releases the car associated with an expired rental and makes it available.
  • Fines Button: Opens a form displaying fines related to this rental.
  • Print Button: Prints all rentals.
  • Exit Button: Closes the application.

General Notes

  • Ensure a rental row is selected before using any button that requires selection.
  • Payment button is only available in specified cases.
  • Use the filter to easily find expired or current rentals.
