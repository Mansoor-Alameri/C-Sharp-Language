مساعدة شاشة إضافة إيجار

هذه الشاشة مخصصة لإضافة إيجار جديد للنظام، حيث يمكن إدخال بيانات الإيجار الأساسية مثل العميل، السيارة، رقم العقد، الخصم، سعر الإيجار اليومي، السعر الكلي، تاريخ البداية والنهاية، وعدد الأيام.

أولاً: الحقول الأساسية

  • المعرف: للقراءة فقط، يتم توليده من قاعدة البيانات.
  • العميل: اختيار العميل من قاعدة البيانات.
  • السيارة: اختيار السيارة المتاحة.
  • رقم العقد الموقع من العميل: الرقم الخاص بالعقد.
  • الخصم: قيمة الخصم على الإيجار.
  • سعر الإيجار اليومي: سعر إيجار السيارة لليوم الواحد.
  • السعر الكلي: مجموع السعر بعد الحساب.
  • تاريخ بداية الإيجار: اليوم الذي يبدأ فيه الإيجار.
  • تاريخ نهاية الإيجار: اليوم الذي ينتهي فيه الإيجار.
  • عدد الأيام: يتم حسابه من تاريخ البداية والنهاية.

ثانياً: الأزرار

  • زر حفظ: لحفظ بيانات الإيجار المدخلة.
  • زر المساعدة: لعرض هذه الصفحة التعليمية.
  • زر الرجوع: للعودة إلى شاشة إدارة الإيجارات.

ملاحظات عامة

  • تأكد من إدخال جميع البيانات المطلوبة قبل الحفظ.
  • المعرف يتم توليده تلقائياً ولا يمكن تغييره.
  • السعر الكلي وعدد الأيام يتم حسابهما تلقائياً.

Add Rental Screen Help

This screen is used to add a new rental to the system, where you can enter the rental's basic information such as client, car, contract number, discount, daily rental price, total price, start and end dates, and number of days.

First: Basic Fields

  • Rental ID: Read-only, auto-generated from the database.
  • Client: Select the client from the database.
  • Car: Select the available car.
  • Contract Number: Number signed by the client.
  • Discount: Discount applied to the rental.
  • Daily Rental Price: Rental price per day.
  • Total Price: Total rental cost after calculation.
  • Start Date: Rental start date.
  • End Date: Rental end date.
  • Number of Days: Calculated from start and end dates.

Second: Buttons

  • Save Button: Saves the entered rental data.
  • Help Button: Displays this help page.
  • Back Button: Returns to the rental management screen.

General Notes

  • Ensure all required data is entered before saving.
  • Rental ID is auto-generated and cannot be changed.
  • Total price and number of days are calculated automatically.
