مساعدة شاشة إدارة / تعديل عميل

هذه الشاشة مخصصة لتعديل بيانات عميل موجود في النظام، حيث يمكن تعديل معظم البيانات الأساسية للعميل مثل الاسم، العنوان، رقم الهاتف، نوع العميل، البطاقة التعريفية، ورخصة القيادة، وغيرها.

أولاً: الحقول الأساسية

  • ID العميل: للقراءة فقط، لا يمكن تغييره.
  • الاسم: يمكن تعديله.
  • رخصة القيادة: يمكن تعديل الرقم.
  • العنوان: يمكن تعديله.
  • نوع الضمانة: غير قابل للتعديل في حالات معينة عدا ذلك يمكن تعديله، تظهر "بدون ضمانة" إذا لم يكن هناك ضمانة.
  • نوع العميل: يمكن تغييره.
  • نوع البطاقة التعريفية: يمكن تغييره.
  • قيمة البطاقة التعريفية: يمكن تعديلها.
  • الجنس: يمكن تغييره.
  • رقم الهاتف الأول: إجباري.
  • رقم الهاتف الثاني: اختياري.
  • تاريخ الميلاد: السنة فقط.
ملاحظة: إذا كان لدى العميل مخالفات غير مدفوعة أو عملية إيجار جارية، قد يتم تقييد بعض الحقول وعدم السماح بتعديلها.

ثانياً: الأزرار

  • زر الرجوع: للعودة إلى شاشة إدارة العملاء.
  • زر الحفظ: لحفظ التعديلات على بيانات العميل.
  • زر المساعدة: لعرض هذه الصفحة التعليمية.
  • زر التقارير: يعرض جميع التقارير المتعلقة بالعميل وبياناته.

ملاحظات عامة

  • تأكد من إدخال البيانات الصحيحة قبل الحفظ.
  • رقم الهاتف الأول يجب أن يكون موجوداً.
  • لا يمكن تعديل الضمانة إذا كان العميل لديه مخالفات غير مدفوعة أو عملية ايجار جارية.




Manage / Edit Client Screen Help

This screen is designed to edit an existing client's data in the system, where most basic fields can be modified such as name, address, phone number, client type, ID type, driving license, and more.

First: Basic Fields

  • Client ID: Read-only, cannot be changed.
  • Full Name: Can be edited.
  • Driving License: Can be modified.
  • Address: Editable.
  • Guarantee Type: Not editable in some Cases Otherwise, it can be editable; shows "No Guarantee" if none.
  • Client Type: Can be changed.
  • ID Type: Can be changed.
  • ID Value: Editable.
  • Gender: Can be changed.
  • Phone 1: Required.
  • Phone 2: Optional.
  • Birth Year: Year only.
Note: If the client has unpaid fines or an ongoing rental, some fields may be restricted and cannot be edited.

Second: Buttons

  • Back Button: Returns to the customer management screen.
  • Save Button: Saves the changes made to the client's data.
  • Help Button: Displays this help page.
  • Reports Button: Displays all reports related to the client and their data.

General Notes

  • Ensure that all data is correct before saving.
  • Phone 1 must be filled.
  • Guarantee type cannot be modified if the client has no Paymented Violations or has a Rental Ongoing.
