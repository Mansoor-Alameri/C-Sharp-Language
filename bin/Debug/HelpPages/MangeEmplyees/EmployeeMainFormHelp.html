مساعدة شاشة إدارة الموظفين

هذه الشاشة مخصصة لإدارة جميع الموظفين داخل النظام، حيث يمكن عرض جميع الموظفين، البحث عن موظف معين، إضافة موظفين جدد، تعديل بيانات الموظفين، وطباعة التقارير.

أولاً: جدول عرض الموظفين

يعرض هذا الجدول جميع الموظفين المسجلين في النظام، ويمكن استخدامه أيضاً لعرض نتائج البحث.

عملية البحث

  • يمكن البحث عن موظف باستخدام المعرف أو الاسم أو البطاقة القومية.
  • اضغط زر بحث لعرض النتائج في الجدول.
  • زر إعادة التعيين يعيد الجدول إلى عرض جميع الموظفين بعد البحث.

ثانياً: الأزرار الأساسية

  • زر المساعدة: لعرض هذه الصفحة التعليمية.
  • زر إضافة موظف: لفتح شاشة إدخال موظف جديد.
  • زر إدارة موظف: لتعديل بيانات الموظف المحدد في الجدول.
  • زر الطباعة: لطباعة قائمة بجميع الموظفين.
  • زر الخروج: لإغلاق البرنامج والخروج.

ملاحظات عامة

  • تأكد من تحديد موظف قبل استخدام زر إدارة الموظف.
  • زر البحث يسهل العثور على موظف محدد بسرعة.
  • يجب حفظ أي تعديلات قبل الخروج من البرنامج.

Employee Management Screen Help

This screen is used to manage all employees in the system, where you can view all employees, search for a specific employee, add new employees, edit employee data, and print reports.

First: Employees Table

This table displays all registered employees in the system, and can also be used to show search results.

Search Function

  • You can search for an employee using ID, Name, or National ID.
  • Click the Search button to display results in the table.
  • The Reset button restores the table to show all employees after a search.

Second: Main Buttons

  • Help Button: Displays this help page.
  • Add Employee Button: Opens the Add Employee screen.
  • Edit Employee Button: Edit the selected employee in the table.
  • Print Button: Prints a list of all employees.
  • Exit Button: Closes the application.

General Notes

  • Ensure an employee is selected before using the Edit Employee button.
  • The search button helps quickly locate a specific employee.
  • Any changes should be saved before exiting the application.
