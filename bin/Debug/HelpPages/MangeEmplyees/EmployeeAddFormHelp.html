مساعدة شاشة إضافة موظف

هذه الشاشة مخصصة لإضافة موظف جديد إلى النظام، حيث يمكن إدخال جميع البيانات الأساسية للموظف مثل المعرف، الاسم، الرتبة، اسم المستخدم، كلمة المرور، تاريخ التوظيف، الراتب، نوع وقيمة المعرف القومي، العنوان، الجنس، أرقام الهواتف، وتاريخ الميلاد.

أولاً: الحقول الأساسية

  • المعرف: للقراءة فقط، يتم توليده تلقائياً من قاعدة البيانات.
  • الاسم: الاسم الأول والاخير للموظف.
  • رتبة الموظف: موظف أو مدير.
  • اسم المستخدم: اسم المستخدم الخاص بالدخول للنظام.
  • كلمة المرور: كلمة المرور الخاصة بالنظام.
  • تاريخ التوظيف: تاريخ بدء عمل الموظف.
  • الراتب: الراتب الشهري للموظف.
  • نوع المعرف القومي: مثل بطاقة قومية أو جواز سفر.
  • قيمة المعرف القومي: الرقم الفعلي للمعرف.
  • العنوان: عنوان سكن الموظف.
  • الجنس: ذكر أو أنثى.
  • رقم الهاتف الأول: رقمي وأجباري.
  • رقم الهاتف الثاني: اختياري.
  • تاريخ الميلاد: يوم/شهر/سنة ميلاد الموظف.

ثانياً: الأزرار

  • زر حفظ: لحفظ بيانات الموظف المدخلة.
  • زر المساعدة: لعرض هذه الصفحة التعليمية.
  • زر الرجوع: للعودة إلى شاشة إدارة الموظفين.

ملاحظات عامة

  • تأكد من إدخال جميع البيانات المطلوبة قبل الحفظ.
  • المعرف يتم توليده تلقائياً ولا يمكن تغييره.
  • رقم الهاتف الأول أجباري ويجب إدخاله بشكل صحيح.

Add Employee Screen Help

This screen is used to add a new employee to the system, where you can enter all essential employee data such as ID, name, rank, username, password, hiring date, salary, national ID type and value, address, gender, phone numbers, and date of birth.

First: Basic Fields

  • ID: Read-only, auto-generated from the database.
  • Name: First and last name of the employee.
  • Employee Rank: Employee or Manager.
  • Username: Username for system login.
  • Password: Password for the system.
  • Hiring Date: Date the employee started work.
  • Salary: Monthly salary of the employee.
  • National ID Type: e.g., National Card or Passport.
  • National ID Value: The actual ID number.
  • Address: Employee's residential address.
  • Gender: Male or Female.
  • Phone Number 1: Mandatory primary phone number.
  • Phone Number 2: Optional secondary phone number.
  • Date of Birth: Employee's birth date.

Second: Buttons

  • Save Button: Saves the entered employee data.
  • Help Button: Displays this help page.
  • Back Button: Returns to the Employee Management screen.

General Notes

  • Ensure all required data is entered before saving.
  • ID is auto-generated and cannot be changed.
  • Phone Number 1 is mandatory and must be entered correctly.
