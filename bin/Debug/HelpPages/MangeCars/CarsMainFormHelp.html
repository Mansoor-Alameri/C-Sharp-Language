مساعدة شاشة إدارة السيارات

هذه الشاشة مخصصة لإدارة جميع السيارات داخل النظام، حيث يمكن عرض جميع السيارات، البحث عن سيارة معينة، إضافة سيارات جديدة، تعديل بيانات السيارات، وطباعة التقارير.

أولاً: جدول عرض السيارات

يعرض هذا الجدول جميع السيارات المسجلة في النظام، ويمكن استخدامه أيضاً لعرض نتائج البحث.

عملية البحث

  • يمكن البحث عن سيارة باستخدام اسم السيارة أو المعرف.
  • اضغط زر بحث لعرض النتائج في الجدول.
  • زر إعادة التعيين يعيد الجدول إلى عرض جميع السيارات بعد البحث.

ثانياً: الأزرار الأساسية

  • زر المساعدة: لعرض هذه الصفحة التعليمية.
  • زر إضافة سيارة: لفتح شاشة إدخال سيارة جديدة.
  • زر إدارة سيارة: لتعديل بيانات السيارة المحددة في الجدول.
  • زر الطباعة: لطباعة قائمة بجميع السيارات.
  • زر الخروج: لإغلاق البرنامج والخروج.

ملاحظات عامة

  • تأكد من تحديد سيارة قبل استخدام زر إدارة السيارة.
  • زر البحث يسهل العثور على سيارة محددة بسرعة.
  • يجب حفظ أي تعديلات قبل الخروج من البرنامج.

Car Management Screen Help

This screen is used to manage all cars in the system, where you can view all cars, search for a specific car, add new cars, edit car data, and print reports.

First: Cars Table

This table displays all registered cars in the system, and can also be used to show search results.

Search Function

  • You can search for a car by Car Name or Car ID.
  • Click the Search button to display results in the table.
  • The Reset button restores the table to show all cars after a search.

Second: Main Buttons

  • Help Button: Displays this help page.
  • Add Car Button: Opens the Add Car screen.
  • Edit Car Button: Edit the selected car in the table.
  • Print Button: Prints a list of all cars.
  • Exit Button: Closes the application.

General Notes

  • Ensure a car is selected before using the Edit Car button.
  • The search button helps quickly locate a specific car.
  • Any changes should be saved before exiting the application.
