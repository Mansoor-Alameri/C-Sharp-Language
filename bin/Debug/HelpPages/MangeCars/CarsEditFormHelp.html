مساعدة شاشة تعديل سيارة

هذه الشاشة مخصصة لتعديل بيانات سيارة موجودة في النظام، حيث يمكن تعديل جميع الحقول بنفس الحقول الموجودة في شاشة إضافة سيارة.

أولاً: الحقول الأساسية

  • المعرف: للقراءة فقط، يتم جلبه من قاعدة البيانات ولا يمكن تغييره.
  • البراند: تعديل اسم الشركة المصنعة للسيارة.
  • الموديل: تعديل موديل السيارة.
  • الفئة: تعديل فئة السيارة (SUV، سيدان، إلخ).
  • نوع الجير: تعديل نوع الجير (أوتوماتيك أو يدوي).
  • نوع الوقود: تعديل نوع الوقود (بنزين، ديزل، كهرباء، إلخ).
  • سعر الإيجار اليومي: تعديل السعر اليومي للإيجار.
  • سنة الصنع: تعديل سنة تصنيع السيارة.
  • سعر السيارة: تعديل السعر الكامل للسيارة.
  • اللون: تعديل لون السيارة.

ثانياً: الأزرار

  • زر حفظ: لحفظ التعديلات على بيانات السيارة.
  • زر المساعدة: لعرض هذه الصفحة التعليمية.
  • زر الرجوع: للعودة إلى شاشة إدارة السيارات.
  • زر التقارير: يعرض جميع إيجارات السيارة ومخالفاتها.

ملاحظات عامة

  • تأكد من إدخال البيانات الصحيحة قبل الحفظ.
  • المعرف لا يمكن تغييره ويجب أن يبقى كما هو.
  • زر التقارير مفيد للاطلاع على جميع البيانات المرتبطة بهذه السيارة.

Edit Car Screen Help

This screen is used to edit an existing car in the system, where all fields are editable just like in the Add Car screen.

First: Basic Fields

  • ID: Read-only, fetched from the database and cannot be changed.
  • Brand: Edit the car manufacturer.
  • Model: Edit the car model.
  • Category: Edit car category (SUV, Sedan, etc.).
  • Gear Type: Edit gear type (Automatic or Manual).
  • Fuel Type: Edit fuel type (Petrol, Diesel, Electric, etc.).
  • Daily Rental Price: Edit daily rental price.
  • Manufacturing Year: Edit car manufacturing year.
  • Car Price: Edit total car price.
  • Color: Edit car color.

Second: Buttons

  • Save Button: Saves the changes to the car data.
  • Help Button: Displays this help page.
  • Back Button: Returns to the car management screen.
  • Reports Button: Displays all rentals and violations related to this car.

General Notes

  • Ensure all data is correct before saving.
  • ID cannot be changed and must remain the same.
  • The Reports button is useful to view all information related to this car.
